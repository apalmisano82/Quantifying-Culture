###### CONTENTS ######

#Folder "dataset"
* A .csv file providing the dataset (pottery.csv).


#Folder "images"
* Pictures of Khabur Ware potsherds stored in the British Museum (1.PNG, 2.PNG,...) 
* A .txt file providing the metadata of the images(metadata_pottery.txt).

#Folder GIS
* A GeoJSON file of archaeological sites yielding Khabur Ware (khabur_sites.geojson)
* A .txt file providing the metadata of the sites yielding Khabur Ware (khabur_sites.txt)
